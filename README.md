## Arch Linux Desktop Setup

Este repositorio contiene mi configuración personalizada de entorno de escritorio en Arch Linux, enfocada en rendimiento, minimalismo y productividad mediante un enfoque *tiling window manager*.

### Componentes principales

- **Window Manager:** Awesome (tiling window manager altamente configurable en Lua)  
- **Terminal:** Alacritty (rápido, ligero y acelerado por GPU)  
- **Barra de estado:** Polybar (modular y altamente personalizable)  
- **Launcher de aplicaciones:** Rofi (rápido y minimalista)  
- **Compositor:** Picom (para transparencias y efectos visuales)  
- **Shell:** Zsh con Powerlevel10k (`.p10k.zsh`)  
- **Inicio de sesión:** `.xprofile` para configuración de entorno al iniciar X  

### Características

- Configuración minimalista y eficiente  
- Flujo de trabajo basado en teclado  
- Estética limpia con transparencias y efectos suaves  
- Alta personalización del entorno
